Formula Reference

DDRAM Usage Calculations:

Usage % = ((Total - Available) / Total) × 100
Usage MB = (Usage % / 100) × Total Memory

CPU Load Calculations:

Load % = ((1024 - Available) / 1024) × 100
Idle % = 100 - Load %
